H2GO

** Won Best Design Award during project demo day **


H2GO is a mock e-commerce site built for selling water-based products. Users can just browse products online, or create an account to add/remove items to their cart using a SQL database. Auth0 handles authentication, and the Stripe API has been implemented to handle transactions.


Built with React, Redux, Express, Node, and SQL.

  • GitHub: SOURCE CODE

Q2- Communications Platform

The Q2 is an interactive platform for students and mentors to communicate. Has built in functionality that allows instructors to manage their students and keep track of relevant student data in the form of charts. Q2 allows for users to ask and answer questions in real-time using websockets.


Built with React, Redux, Express, NodeJS, socketIO, ChartsJS, and MomentJS.

  • Demo: LIVE WEBSITE
  • GitHub: SOURCE CODE

Polymath (Work In Progress)

Like Wikipedia meets Khan Academy. Polymath is a user-driven social community where you can learn anything for free. Powered by users, search any topic and you'll be guided to external resources that will give you the information you need.


Built with React, Redux, GraphQL, D3, Express, NodeJS, postgreSQL, and TypedJS.
